Presentation - Modern Perl Object-Oriented Programming

Tags: meeting

Posted on 2014-10-07

This month's presentation will be on modern Perl OO and the tools to make object-oriented code easy.

Topics will include

  • Simple objects
  • Inheritance and roles
  • Method modifiers
  • Type checking and coercion
  • Meta classes

CPAN modules covered:

  • Moo
  • Role::Tiny
  • Type::Tiny
  • Moose

Doors open at 6:30pm. Meeting starts at 7:00pm. RSVP on the Chicago.PM Meetup
