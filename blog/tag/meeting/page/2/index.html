Meeting - FFI::Raw - libffi bindings for Perl

Tags: meeting

Posted on 2014-06-02

This month, Dylan Cali from Bank of America will be giving a talk on FFI::Raw and his ZeroMQ Perl binding that uses it: ZMQ::FFI.

FFI stands for Foreign Function Interface, and it is another way to access native libraries from dynamic languages. FFI::Raw provides an API on to libffi, allowing you to call native library functions with pure Perl code!

RSVP on the Chicago.PM Meetup page
